# The first ---- 使用 git 上传东西到 github 上

### 1. 克隆自己`github` 上新建的仓库地址

```shell
git clone https://github.com/zlp-zzZ/Web-Study.git 
# 这里是克隆你自己的地址 ^:^
```



### 2.  进入到你克隆下来的项目根目录下

```shell
cd Web-Study
# 也可以找到克隆目录的位置，shift + 鼠标右键 打开命令行窗口
```

### 3. next

```shell
git add .
```

### 4. next 

```shell
git commit -m "XXXXXXX"
# 填写你要对此次修改的描述 可以不写
```

### 5. next

```shell
git push -u origin master
# 会让你输入你的github 名字 密码
```
